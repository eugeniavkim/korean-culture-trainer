두벌식 keyboard reference
Standard Korean Keyboard — 두벌식 (Dubeolsik)
Enable Korean on iPhone: Settings → General → Keyboard → Keyboards → Add New Keyboard → Korean (2-Set) → tap 🌐 to switch
Enable on Windows: Settings → Time & Language → Language → Add Korean → Alt+Shift to toggle
두벌식 rule: Consonants (자음) on the left side · Vowels (모음) on the right side
Example: 한국 → ㅎ+ㅏ+ㄴ+ㄱ+ㅜ+ㄱ → type: g k s r n r
Timer pace guide: 🌱 Easy = 0.9s/char  ·  ⚡ Medium = 0.55s/char  ·  🔥 Hard = 0.30s/char
Color goes green → yellow → red as you use up your time budget for that phrase.
